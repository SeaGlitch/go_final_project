# Файлы итогового проекта курса "Go-разработчик с нуля"

В данном задании реализован функционал простейшего планировщика задач с базой данных SQL.

В директории `tests` находятся тесты для проверки API, которое реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

Веб-сервер запускается по адресу http://localhost:7540/. Порт можно поменять в настройках окружения (.env).
Из дополнительных заданий со звездочкой реализовано получение данных из окружения (.env), все правила повторения, поиск задач, докер-образ.
Пока не реализован запрос токена.

Параметры для тестов:

var Port = 7540
var DBFile = "../database/scheduler.db"
var FullNextDate = true
var Search = true
var Token = ``

Версия Go 1.22.5
Команды для запуска в докере: 
docker build -t my_go_project .
docker run -d -p 7540:7540 my_go_project
